Program description: 
Our program is a flight reservation management system that will let users book, reserve, search,and update flights that they will want to take. 

Steps:

1. In flights tab you will be able to pick starting destination, end destination and date of flight.

2. Pick a flight that will suit your schedule and it will appear on the right side of the application

3. Please enter AirLine, Name, and Citizenship and click make reservation

4.In reservations tab you will be able to search for reservations that you have made 

5.In the bottom part of the app just enter name, code, or the airline to search for reservation

6.To update a reservation you will need to choose which reservation to update and once you do their will be options on updating the reservation on the right side of the application

Run program: Command prompt command: java -jar Assignment2.jar
